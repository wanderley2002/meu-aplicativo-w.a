🚀 Meu Aplicativo W.A (Ranking & Performance)
Este é um ecossistema de monitoramento e interação social em tempo real, desenvolvido para demonstrar alta performance em sistemas de Ranking e Comunicação.
🛠 Tecnologias & Infraestrutura
Back-end & Real-time
Node.js & Express: Arquitetura de servidor robusta e escalável.
Socket.io: Implementação de comunicação bi-direcional (WebSockets) para o Chat Online e atualizações de Ranking ao vivo, sem necessidade de recarregar a página.
Banco de Dados & Persistência
TiDB Cloud (MySQL): Banco de dados relacional distribuído na nuvem, garantindo a integridade dos dados e suporte a múltiplos acessos simultâneos.
Ambiente & Deploy
Render & GitHub: Pipeline de CI/CD configurado para que cada atualização no código seja refletida automaticamente no site.
🏆 Destaques do Projeto
Chat Global em Tempo Real: Sistema de mensagens instantâneas utilizando WebSockets para comunicação fluida.
Ranking Dinâmico: Algoritmo que calcula e ordena a performance/consumo dos usuários instantaneamente.
API de Dados: Integração com banco de dados remoto para persistência segura de mensagens e pontuações.
Interface Responsiva: Design focado em usabilidade, adaptado para qualquer tipo de tela (Mobile e Desktop).
